# e-commerse-website-electronic-gadgets
Electronics Store - E-commerce Website

Overview

This is a simple e-commerce website for electronics where users can browse products, add them to their cart, and proceed to purchase. The website includes navigation between different sections such as Home, Products, Cart, and Contact.

Features

Navigation Bar: Allows users to navigate between different sections.

Product Listing: Displays electronic gadgets with images, names, and prices.

Add to Cart: Users can add multiple products to the cart.

Cart Management: Users can view their cart, remove items, and see the total price.

Buy Option: Clicking "Buy Now" will display a success message upon checkout.

Usage

Open the website in a browser.

Navigate to the "Products" section to view available gadgets.

Click the "Add to Cart" button on desired products.

Click on the "Cart" section in the navigation bar to view selected items.

Remove items if needed or click "Buy Now" to complete the purchase.

A success message will appear when the purchase is completed.
